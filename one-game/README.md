# 一号游戏：iPhone 全屏安装

这个目录是“一号游戏”的可安装 Web App 启动器。

## 第一次使用

1. 使用 iPhone Safari 打开 GitHub Pages 中的 `one-game/` 地址。
2. 点击“选择游戏 HTML”。
3. 选择已经下载到“文件”App中的游戏 HTML。
4. 点击“导入并启动”。游戏文件只保存在当前手机的 IndexedDB 中，不上传到服务器。
5. 回到 Safari 分享菜单，选择“添加到主屏幕”，并确认“作为网页 App 打开”。
6. 以后直接点击桌面上的“一号游戏”图标。

## 全屏与锁定

- 从主屏幕图标启动后，以 standalone 模式运行，不显示 Safari 地址栏和底部工具栏。
- 启动器会在进入游戏时请求 Screen Wake Lock，尽量防止屏幕自动熄灭。
- 网页无法禁止系统退出。如需把 iPhone 锁定在游戏中，请开启：设置 → 辅助功能 → 引导式访问。启动游戏后连按三次侧边按钮。

## 更换版本

游戏运行后点击右上角半透明菜单按钮，可以重新导入新的 HTML 版本或清除本地版本。
